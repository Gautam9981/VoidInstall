# VoidInstall
This is based off of archinstall (https://github.com/archlinux/archinstall)

A modular, TUI-based installer for Void Linux, inspired by archinstall.

Features:

Disk partitioning and encryption (LUKS, no LVM)
Filesystem setup
Base system installation (xbps)
User and bootloader setup
Network, graphics drivers, and desktop environment selection
Usage
Run with Python 3. Requires root privileges and Void Linux live environment.
